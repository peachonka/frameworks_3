# Практическая работа 3

Проект показывает работу с настройками из трёх источников, проверку корректности настроек до запуска, базовую защиту для запросов из браузера, ограничение частоты запросов и добавление защитных заголовков.

## Запуск приложения

### Development режим (по умолчанию)

```bash
dotnet run
```

### Production режим (пример)

```bash
dotnet run -- -m Production -r 200 -b 50 --origins "https://myapp.com"
```

**Параметры:**
| Ключ | Назначение | Пример |
|------|------------|--------|
| `-m` | Режим работы | `Development` / `Production` |
| `-r` | Лимит запросов в минуту | `200` |
| `-b` | Burst-лимит | `50` |
| `--origins` | Доверенные источники (через пробел) | `"https://site1.com https://site2.ru"` |

### Через переменные окружения

```bash
$env:NOTES_API_App__Mode = "Production"
$env:NOTES_API_App__RateLimit__RequestsPerMinute = "300"
dotnet run
```
